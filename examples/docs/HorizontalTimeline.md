## Timeline 横向时间轴
用于展示时间轴数据，支持拖动和滚轮。

### 基础用法

::: warning

```vue demo
<template>
aaa
</template>
```

### Attributes

| 参数        | 说明           | 类型    | 可选值                                             | 默认值 |
| ----------- | -------------- | ------- | -------------------------------------------------- | ------ |
| list        | 时间轴列表数据       | object  | —                                                 | —      |

### Slots

| 插槽名        | 说明           |
| ----------- | -------------- |
| ——          | 内嵌的列表卡片  |
| time        | 时间轴横线      |
