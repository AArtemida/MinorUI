## 快速上手

本节将介绍如何在项目中使用 minor-ui。

### 全局导入

```js
// 导入ui组件库
import minorUI from 'minor-ui'

// 导入UI样式
import 'minor-ui/lib/themes/index.css'

// 注册组件库
Vue.use(minorUI)
```
